# Portafolio Profesional de Soluciones Digitales - Marx Echeverría

Este es un portafolio de **alto nivel** diseñado para proyectar autoridad, profesionalismo y excelencia técnica. Está estructurado para diferenciar claramente entre sistemas complejos en producción real y arquitecturas de concepto/demo.

## 💼 Estructura del Portafolio

El portafolio se organiza en dos categorías principales:

### 1. Sistemas en Producción (Real-world Apps)
Plataformas transaccionales y herramientas administrativas operativas:
- **MegaTienda:** E-commerce tecnológico con inventario y analítica (PHP/MySQL).
- **Sistema de Rifas:** Gestión transaccional de boletos y pagos (API REST).
- **Phindol Swimwear:** Tienda visual de alta conversión.
- **DAYENU SPA:** Plataforma de bienestar y reservas (React/Tailwind).
- **Fundación Imaynalla:** CMS autogestionable (WordPress).
- **Registro de Niños:** Sistema administrativo CRUD interno.

### 2. Arquitecturas de Concepto (UI/UX Demos)
Demos técnicas que demuestran dominio en diseño y frontend modular:
- **Echeverría Legal:** Consultoría Legal Premium.
- **CardioElite Medical:** Salud Digital y UX Médica.
- **Nexus Systems:** Innovación y Ciberseguridad.
- **Suite Educativa:** Juegos de lógica en tiempo real.
- **Terminal Interactiva:** Portafolio creativo basado en consola.

## 🎨 Concepto Visual: "Executive Dark"
- **Diseño:** Minimalismo serio con enfoque en tipografía editorial (`DM Serif Display` & `Inter`).
- **Paleta:** Negro profundo, carbón y acentos en **Oro Refinado** (`#c5a059`).
- **Interactividad:** Previsualizaciones dinámicas (iframes) integradas en mockups de navegador realistas.
- **Navegación:** Sistema de filtrado fluido para categorizar proyectos.

## 🛠️ Stack Tecnológico
- **Core:** HTML5, CSS3, JavaScript (ES6+).
- **Frameworks/Libs:** Tailwind CSS, Animate On Scroll (AOS), Font Awesome 6.
- **Arquitectura:** Estructura modular de archivos por proyecto para fácil mantenimiento.
