# Changelog

## [2.1.0] - 2026-04-07

### Añadido
- **Executive Placeholders**: Implementación de previsualizaciones estáticas de alto impacto para sitios externos con restricciones de seguridad (`X-Frame-Options`).
- **Restauración de Proyectos**: Reintegración de Raíces & Futuro (Opción 4) y Nicky Shop (Opción 5) al grid principal.
- **Centrado Absoluto**: Refinamiento del posicionamiento de previsualizaciones (iframes) para un centrado horizontal y vertical perfecto.

### Cambiado
- **Mantenibilidad**: Limpieza de estructura HTML y corrección de etiquetas mal cerradas en el grid de proyectos.
- **UX de Navegación**: Ajuste de escalas y efectos hover para una interacción más fluida en el catálogo.

## [2.0.0] - 2026-04-07

### Añadido
- **Rediseño Completo "Executive Dark"**: Interfaz profesional de alto impacto orientada a perfiles ejecutivos y corporativos.
- **Nuevos Proyectos en Producción**: Integración de MegaTienda, Sistema de Rifas, Phindol Swimwear, DAYENU SPA, Fundación Imaynalla y Registro de Niños.
- **Sistema de Filtrado**: Navegación dinámica entre "Todos", "Producción" y "Conceptos".
- **Mockups de Navegador**: Las previsualizaciones dinámicas (iframes) ahora se presentan dentro de marcos de navegador realistas.
- **Tipografía Editorial**: Implementación de `DM Serif Display` para títulos de impacto.

### Cambiado
- **Estructura de index.html**: Reorganización total del layout para manejar más de 10 proyectos de forma ordenada.
- **Paleta de Colores**: Cambio de tonos pastel a una estética oscura con acentos dorados para mayor seriedad.
- **UX de Previsualización**: Ajuste de escalas en iframes para una visualización más profesional y centrada.

## [1.1.0] - 2026-04-07
- Reestructuración modular inicial y diseño minimalista pastel.

## [1.0.1] - 2026-03-28
- Conversión de PHP a HTML y soporte para GitHub Pages.
